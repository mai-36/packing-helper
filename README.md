# packing-helper
旅行専用のパッキングリスト共有アプリです

## サービス概要
「あの国、何持っていったっけ？」を解決する、海外・国内旅行専用の「行き先別」パッキングリスト共有サービスです。 過去の自分の持ち物リストを保存し、次回の旅行時にチェックリストとして再利用できます。

## このサービスへの思い・作りたい理由
私は国内外問わず旅行が大好きですが、「コンセントの変換プラグは用意していたのに充電コード自体がない」「現地の通貨は、あの場所で換金していたらお得だったのに」「思ったより薬を飲む機会があり、常備薬が現地で切れてしまった」「到着すると予想よりもかなり寒かった.でも羽織を持ってこなくて思わぬ出費...」などといった場面が多々あり、現地で後悔していました。 既存のToDoアプリでは日常の用事と混ざってしまうため、旅行の「行き先」や「季節」に紐づいた、旅行者のための専用リスト管理ツールが欲しいと考えたのがきっかけです。

## ユーザー層について
頻繁に旅行に行く人：準備を効率化し、忘れ物をゼロにしたい人。
初めての場所へ行く旅行初心者：ベテラン旅行者が「何を持っていったか」を参考にしたい人。

## サービスの利用イメージ
ユーザーは「タイ（夏・3泊4日）」のようにタイトルをつけて持ち物リストを作成します。 出発前にスマホでチェックを入れながら準備し、旅行が終わった後は「次回への反省（次はこれがいらなかった等）」をメモして保存。次の旅行ではそのリストをコピーして使い回せます。

## ユーザーの獲得について
旅行系ハッシュタグ（#海外旅行 #パッキング）でSNSに自分のリストをシェア。
「これさえあれば安心」という必須アイテムリストを公開し、旅行コミュニティへ発信。

## サービスの差別化ポイント・推しポイント
「忘れた」を「次に活かす」仕組み：単なるメモではなく、旅行後の「反省メモ」を残せるため、使うほどリストの精度が上がります。
軽量・シンプル：多機能な旅行アプリと違い、パッキングだけに特化しているため、通信環境の悪い海外でもサクサク動きます。

## 機能候補
### MVPリリース（必須機能）
* ユーザー登録・ログイン機能
* 旅行リストの作成・編集・削除（タイトル、行き先、メモ）
* 持ち物アイテムの登録・削除・チェック機能（完了したものを消し込む）

### 本リリース（追加機能）
* リストのコピー機能（過去のリストを基に新しいリストを作る）
* 公開リストの閲覧・検索機能（他の人の持ち物を参考にする）
* カテゴリ分け機能（衣類、電子機器、貴重品など）
